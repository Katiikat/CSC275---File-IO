Assignment - File I/O

Assignment Requirements: 
Read in the contents of the CPP_Reference.txt file you created in the Assignment: Build your own C++ Reference.
Count the number of keywords and the number of reserved words in the file. 
Write the results to a new file called CPP_Enhanced_Reference.txt and to the screen in the order of:
C++ Keywords Defined = X
C++ Reserved Words Defined = X
Write out each Keyword and definition.
Write out each Reserved word and definition.
Format the target textfile like this:
C++ Keywords Defined = 3

asm - Declaration of an inline assembly block
auto - auto placeholder type specifier
bool - data type which only holds values true or false

C++ Reserved Words Defined = 1

and - an alternative for &&
